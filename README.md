# UNO_Deck_Holder
Holds a standard deck of UNO, Monopoly Go or any card size similar to UNO Cards

This project was inspired by the need for a box due to the rain or water that has rotted the box to nothing. As such, I decided to make a regular container that holds a standard 112 card UNO deck, I had to go to the internet to get inspiration and also get some sketches on what I would want included. I started with the box, which would be a standard box with a cover but then I wanted a slight touchup, which led to the UNO symbol incorporation. Then, I wanted to add a lock as I didn't want the box to just spill when not upright. After around 2 hours of work I was done, below is the final design.
